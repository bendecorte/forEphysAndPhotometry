# forEphysAndPhotometry
This project accompanies the Manuscript NMETH-A56047 (under revision 11/21/2025)
Relevant pre-print :  https://assets-eu.researchsquare.com/files/rs-4249277/v1/edcbec8a-1d6f-4f50-8f1c-f9471fdf4c03.pdf
This repository contains the 3D print files (.stl), info PCBs and code for the Ephys and photometry supportive device described in the above referenced manuscripts. 
The included ~48 minute video file is of an adult mouse performing an operant task with milk reinforcement during simultaneous Ephys and fiber photometry recordings using the supportive device described in the above referenced manuscripts.
