#forEphysAndPhotometry 
This project accompanies the paper "Automated device for permitting free movement during simultaneous photometry and electrophysiology in mice" (Manuscript NMETH-A56047, under revision 11/21/2025)

This repository contains the 3D print files (.stl), info on PCBs, and code for the Ephys and photometry supportive device described in the above referenced manuscript. 
The video files are of an adult mouse performing an operant task with milk reinforcement during simultaneous Ephys and fiber photometry recordings using the supportive device described in the above referenced manuscript.
